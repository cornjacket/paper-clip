## What it covers
- Pitch: open-source orchestration for teams of AI agents. "If [[openclaw|OpenClaw]] is an *employee*, [[paperclip|Paperclip]] is the *company*." A Node.js server + React UI.
- The three-step model: define the goal → hire the team → approve and run. See [[goal-alignment]] and [[org-chart-and-agents]].
- Works with OpenClaw, Claude Code, Codex, Cursor, Bash, HTTP — "if it can receive a heartbeat, it's hired." See [[bring-your-own-agent]] and [[heartbeats]].
- Nine headline features and seven "why it's special" properties.
- "What's Under the Hood": the twelve systems mapped in [[architecture]].
- "What Paperclip is not" — see [[positioning]].
- Quickstart, requirements, dev commands, telemetry — see [[installation-and-setup]].
- A short [[roadmap]] preview and [[community-and-ecosystem|community links]].
