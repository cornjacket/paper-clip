One Paperclip deployment can run an unlimited number of companies with complete data isolation — one control plane for a whole portfolio.

## Details
Every entity is company-scoped, so each company has separate data and audit trails. Backed by [[identity-and-access|company memberships]] and [[secrets-and-storage|company-scoped secrets]]; organizations move between deployments via [[company-portability]].
